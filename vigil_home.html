Vigil Event Tracker


About the app

Vigil Event Tracker is a free app. Used to track latest and upcoming event and get notified so that you will not miss important events and wish your dear ones on their important days, attend meetings on time etc.,


Accessing user data

The app do not store any data as it does not maintain user database. But, the app completely depends on the users own Google Spreadsheet which is used as a database and maintained by user itself. The app will access the data contained in the google sheet for the purpose of event tracking (Notifying the user about the events). No sign-in or login is required as per the apps latest version.


Enhancing User Experience

For enhancing user functionality, an add-on must be installed in the Google Spreadsheet by the user manually. The add-on is a app script required for enabling google sheet as user database and making it accessible to Android's Vigil Event Tracker app. This will make user keep track of their important events over their mobile phone with the help of Vigil App. User will get notified about latest and upcoming events based on the data present in user-maintained Google Spreadsheet
